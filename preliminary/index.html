Focal mecanism analysis by full waveforms

  • Green's function is caluclated using finite difference method in a given structure (mostly 1D).
  • Stations and frequency band are manually selected.
  • Genetic algorithm is applied to get the fault mechanism (strike, dip, rake), timing and moment magnitude.
  • See Aochi and Burnol (2018), Dominique et al. (2022).
